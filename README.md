# Ecwid Checkout Script

This repository hosts a JavaScript file for Ecwid `customJsUrl`.

## Main file
- `checkout-location.js`

## GitHub Pages URL format
- `https://USERNAME.github.io/ecwid-checkout-script/checkout-location.js`

## Notes
- Do not store API keys in frontend JS.
- Use this file as the Ecwid `customJsUrl`.
